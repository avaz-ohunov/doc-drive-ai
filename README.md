# DocDriveAI Frontend

DocDriveAI — это облачное хранилище с функциями автоматического анализа содержимого файлов и тегирования с помощью ИИ

## 🚀 Как запустить Frontend

1. Скачайте [node.js](https://nodejs.org/en/download)
2. Напишите в консоли ```npm install```
3. Напишите в консоли ```npm run dev```

## Как запустить [Backend](https://github.com/pararam2006/dockdriveai)

## Как запустить ИИ-модель

1. Скачайте [Ollama](https://ollama.com/)
2. Напишите в консоли ```ollama serve```
3. Напишите в другой вкладке консоли ```ollama run llama3.2```

## 🏗️ Технологический стек

- Язык программирования: TypeScript
- Библиотека: React 18
- Установщик: Vite 3
- Хранение файлов: Yandex Cloud (S3 Compatible)
- База данных: SQLite
- ИИ: Ollama (llama 3.2)
